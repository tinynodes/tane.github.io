I Think I Found a Privacy Exploit in ChatGPT

 Published: Apr 14, 2023  (last updated: Apr 14, 2023) | ~ 1500 words | ~ 7 minutes reading time
OpenAI ChatGPT LLM AI
 ... comments

> tl;dr: I found that if you pass empty prompts to ChatGPT it spits back results anyway At first I thought they might be hallucinations but now I believe I'm also getting back other people's responses

Earlier last month; to a lot of fanfair of developers - along with enterprise, media, and government interests - OpenAI release their new Large Language Model ChatGPT-4 .

For LLMs it’s definetly very advanced. Just days before I was playing with alpaca.cpp - a model that is designed to run on a CPU with a smaller set of memory constraints. I had started with building a very simple web interface using NodeJS and sockets to parse the command line. Shortly after I started working on this, my GTP-4 invite arrived, and I started playing with the API. Quickly I could see with the right kind of prompts, it can be a useful tool in the right hands. It’s already helped me re-write complex code into simpler methods that pass the tests, or reduce complexity by moving code to functions:

However I started noticing something odd - due to a bug in my code, I was passing empty prompts to the ChatGPT endpoint - but every time I was getting back a response - and they were completly random. From standard “I am an AI model…” responses, to information about people, places and concepts.

The same day, it was #StochasticParrotsDay - an online conference around AI ethics, hype and it’s future - and it gave me an idea - why don’t I turn this into a Mastodon Bot ! (*this was this original URL - it has now since moved to botsin.space )

Since I’ve been running this bot, I’ve come to some conclusions about the responses and I believe that without a prompt a large proportion of the responses are meant for other users. If it’s a bug, possibly passing an unsanatised empty prompt causes it to return something it should not in response.

Of course I could be wrong - these could be stochastic hallucinations being dreamed up by a machine, they could be the API returning a random set of training data that happens to be at the right Entropy - but then ChatGPT itself isn’t then much better than a Markov Chain and OpenAI (and the whole LLM/AI market) have been playing everyone for fools.

But if I am correct then the current OpenAI APIs could be made to potentially leak private or sensitive data, by simply not sanatising their inputs…

The bot will continue to run until at least the end of this month, and all the content will be archive at stochasticparrot.lol .

Setting up the Bot Infrastucture

To get the bot up and running, I wanted to do it for free, and easy to manage. In the end I opted to use GitHub Actions with scheduled tasks to create it - this allowed me to set up a script that ran hourly - calling the ChatGPT API with an empty prompt, and turning it into a toot. I also found that passing only a space character to the Dall-E API also produced images.

With both scripts, after getting a response from OpenAI, I use it to generate one or more toots - depending on the length as a set of replies, with the images first download them and then upload them as attachments first.

Making Polly Speak

Up to this point, I had just been working with text and images - but I had recently seen ElevenLabs in some tech news, and that they had a text-to-speech API. After some initial issues (which used up all of the free credit) - I eventually set up another action that took the OpenAI response, and passed it to the ElevenLabs API - this then provided a MP3 stream of the speech, saved locally and again upload to Mastodon and attach to a toot.

A fun experiment

Overall, writing this bot was a fun experiment - but I probably learned more about writing better pipelines, than I did about AI and LLMs. What did surprise me was how often the responses seem to be to questions that were not asked - where are these responses being generated? Are we seeing the flicker of AGI? Or just the stochastic ramblings of a machine run by some sketchy people .
